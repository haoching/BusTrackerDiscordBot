## setting
`蒿吐開始你的機器人`
### nextcord
- [nextcord](https://pypi.org/project/nextcord/)
#### install nextcord

```
# Linux/macOS
python3 -m pip install -U nextcord

# Windows
py -3 -m pip install -U nextcord
```
### request
```
pip install requests
```
### dotenv
- [dotenv](https://pypi.org/project/python-dotenv/)

####install dotenv
```
pip install python-dotenv
```
